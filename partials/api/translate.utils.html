utils
service in module translate

Description

Service provides some utility functions like errors, warnings and etc.

Methods

  • complexDataTransformation(data)

    Transfrom complex object to flat object. Translations will be available via '.' (see example)

    Parameters
    ParamTypeDetails
    dataObject

    Data

    Returns
    Object

    Transformed Values

    Example

    $translate.config({
     ...
     dataTranformation: $translateUtils.complexDataTransformation
});

    Input:

    {
  "title": "Translator",
  "dashboard": {
    "hello": "Hallo!",
    "menu": {
      "main": "Home"
    }
  }
}

    Output:

    {
  "title": "Translator",
  "dashboard.hello": "Hallo!",
  "dasboard.menu.main": "Home"
}

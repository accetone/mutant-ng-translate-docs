events
service in module translate

Description

Service responsible for providing access to library events. Provide list of event objects and you can subscribe to each (see implementation here).

Dependencies

  • utils

Events

  • allPartsLoaded

    Fires when all registered parts for some language loaded

    Parameters
    ParamTypeDetails
    datastring

    { lang: language key }

  • langChanged

    Fires when language changed

    Parameters
    ParamTypeDetails
    datastring

    { from: old language key, to: new language key }

  • partLoaded

    Fires when any part for any language loaded

    Parameters
    ParamTypeDetails
    datastring

    { part: partname, lang: language key }

  • translationsUpdated

    Fires when translations for any language updated

    Parameters
    ParamTypeDetails
    datastring

    { lang: language key }

Example

var callback = function (data) { console.log(data); }

// how subscribe ?
var token = $tranlslateEvents.langChanged.subscribe(callback);

// how unsibscribe ?
token.unsibscribe();

// how to register one-time callback ?
var token2 = $tranlslateEvents.langChanged.subscribe(callback, true);
